# 📝 Todo App

🚀 Built with Flutter & Firebase
📱 Clean UI & Smooth UX
📊 Productivity Tracking

---

## 📖 About The App

This is a modern Todo App designed to help users manage their daily tasks efficiently.
The app allows users to organize tasks, track their productivity, and view their schedule using a clean and intuitive interface.

Key goals of the app:

* Improve daily productivity
* Provide clear task organization
* Offer insights through statistics

---

## 🚀 Features

* 🔐 Authentication using Firebase
* 🗓️ Calendar View
* ✅ Task Management (CRUD)
* 📊 Statistics Dashboard
* ⏰ Task Scheduling

---

## 🛠️ Tech Stack

* Flutter
* Firebase Authentication
* Cloud Firestore
* Shimmer
* SVG Support
